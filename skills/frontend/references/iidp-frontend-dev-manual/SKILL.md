- 定位：IIDP 前端开发文档索引与事实来源（框架机制/组件/扩展视图/扩展钩子/工程等）。
- 上游：无。
- 下游：[iidp-frontend-spec-doc](../iidp-frontend-spec-doc/SKILL.md)、[iidp-frontend-spec-code](../iidp-frontend-spec-code/SKILL.md)、[iidp-frontend-extension-dev](../iidp-frontend-extension-dev/SKILL.md)、[iidp-frontend-init](../iidp-frontend-init/SKILL.md) 在需要文档事实时应优先跳转到此处。

## 使用指引

1. 需要框架机制速查时，先读 [quick-reference.md](quick-reference.md)。
2. 需要写扩展代码时，读 [iidp-frontend-extension-dev](../iidp-frontend-extension-dev/SKILL.md)，必要时再按本文索引查原始手册。
3. 需要查组件属性时，先读组件规则文件；未覆盖组件再按“组件”索引定位原始文档。
4. 需要工程配置、依赖、启动或按需加载时，优先读 [iidp-frontend-init/reference.md](../iidp-frontend-init/reference.md)。
5. 遇到常见问题、报错或扩展案例时，查“平台常见问题”索引。
6. 索引未覆盖时，再搜索 `iidpDoc/` 目录。
